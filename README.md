# 欢迎阅读算法和数据结构说明 📃

这个项目专注于使用多种编程语言实现不同的数据结构和算法。如果有人在编码时遇到相应的问题，这里应该可以找到你想要的答案。

> ⚠ *为了方便查询的分类，这里的文件夹均以相应的**前缀**加**数据结构名**称或**算法名**称命名，
> 以 **A_** 为前缀的是算法，以 **DS_** 为前缀的是数据结构。*
